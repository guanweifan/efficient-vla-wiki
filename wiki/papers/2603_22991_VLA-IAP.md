## Claim
- 页面定位：这是一篇 **interaction-aligned, training-free visual token pruning** 论文；它主要重写的是 pruning 决策逻辑，而不是通过再训练 backbone 去获得新的表示能力。
- 这篇论文要解决的问题是：现有 VLA visual token pruning 方法大多延续了 **Perception-First** 偏置，主要根据 semantic saliency 或简单 temporal cue 做裁剪，因此容易在任务早期把“视觉上不显眼但对物理操作关键”的结构性区域提前裁掉。作者据此提出 **VLA-IAP**，把 pruning 从被动的 perception filtering 改成 **Interaction-First** 的显式交互对齐机制。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 1-3 页 Abstract、Introduction、Fig. 1。
- 方法上的主张是一个 **training-free** 的 token pruning 框架，由两部分构成：一是 **Geometric Prior Mechanism**，通过 edge enhancement 保留结构锚点；二是 **Interaction-Aligned Dynamic Strategy**，根据 semantic-motion IoU 在 conservative pruning 和 aggressive pruning 之间动态切换。其关键 framing 不是“更好的注意力裁剪”，而是借助几何先验和交互状态判断，让 pruning 强度与 manipulation phase 对齐。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 2-5 页 Fig. 1、Fig. 2、Sec. 3。
- 论文保留了多组 headline numeric claims，但这些结果属于不同层级：
  - **simulation efficiency-performance claim**：在 **LIBERO** 上给出 `97.8% success rate + 1.25× speedup`。
  - **more aggressive pruning claim**：在更激进设置下给出 `1.54× speedup`，但这是另一组 token retention ratio，不应与前一组结果混写。
  - **real-world deployment claim**：在真实机器人部署中报告 `up to 1.48× inference acceleration` 并提升平均操作成功率；这又是新的平台与评估口径。
- 更稳的单篇主命题应写成：VLA-IAP 通过几何先验与 interaction-aligned dynamic strategy，让 pruning 强度更贴近 embodied manipulation 的交互需求；其 headline 数字跨越不同 retention ratio、benchmark、backbone 与 deployment setting，不能压成一个统一无条件的 superiority claim。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 1 页 Abstract；第 9-11 页 Table 2、Table 3、runtime / real-world analysis；第 15-16 页 Conclusion / appendix qualitative analysis。
