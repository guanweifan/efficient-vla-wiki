## Claim
- 页面定位：这是一篇 **unified dual-system foundation VLA** 论文；核心贡献是把 System 1 execution module 嵌入 VLM-based System 2 内部，而不是外接一个独立轻量 policy。
- 这篇论文提出 **Fast-in-Slow (FiS / FiS-VLA)**，目标是同时解决 robotic manipulation 中的两类核心矛盾：**foundation policy 的泛化能力**与**执行频率/实时性**。作者认为现有 dual-system VLA 虽然把慢推理和快执行拆开了，但由于 System 1 通常是外接的轻量 action model，无法充分继承 VLM-based System 2 的互联网规模预训练知识。来源：[[raw/2506_01953_FiS-VLA.pdf]]，第 1-2 页 Abstract、Introduction。
- 核心主张是：把 **System 1 execution module 嵌入到 VLM-based System 2 内部**，通过部分共享参数、异步频率和异构输入模态，形成一个统一 dual-system foundation model，从而在单个基础模型内部实现 reasoning 与 execution 的协调。来源：[[raw/2506_01953_FiS-VLA.pdf]]，第 1 页 Abstract；第 2-4 页 Fig. 1、Fig. 2、Sec. 3。
- 论文进一步主张，System 1 和 System 2 应承担不同角色，因此需要：
  - **heterogeneous modality inputs**
  - **asynchronous operating frequencies**
  - **dual-aware co-training**  
  其中 System 2 负责低频 reasoning，System 1 负责高频 action execution，并特别引入 point cloud 等几何信息支撑精细操作。来源：[[raw/2506_01953_FiS-VLA.pdf]]，第 1-3 页 Abstract、Introduction；第 4-6 页 Sec. 3。
- 作者给出的 headline numeric claim 是：FiS-VLA 相比以往 SOTA 在**simulation 平均成功率提升 8%**、在**real-world 平均成功率提升 11%**，并在 action chunk size = 8 时实现 **117.7 Hz** 控制频率。需要注意，这些数字是论文 headline claim，后续仍需精确钉到对应表格、机器人平台和比较对象。来源：[[raw/2506_01953_FiS-VLA.pdf]]，第 1 页 Abstract；第 3 页 contributions；第 7-9 页 quantitative results 与 appendix。

## Methodology Index
- Fast-in-Slow (FiS) / FiS-VLA
- Unified dual-system VLA
- System 1 in System 2
- Partial parameter sharing
- Intact VLM for System 2 reasoning
- Final transformer blocks repurposed as System 1
- Heterogeneous modality inputs
- Asynchronous operating frequencies
- 2D images + language for System 2
- Robot state + 2D image + 3D point cloud for System 1
- Lightweight 3D tokenizer / point cloud tokenization
- Shared vision encoder for geometric features
- Diffusion-based action generation
- Dual-aware co-training
- Next-token prediction for System 2
- Large-scale robot pretraining (860K trajectories)
- Action chunking / frequency ratio exploration
