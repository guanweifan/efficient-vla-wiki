# Synthesis Page Template

## Question
- 这一页回答的主题问题是什么。

## Shared Ground
- 哪些事实在不同论文之间可以稳定成立。

## Route Split
- 主要路线如何分化，依据是什么。

## Boundary Conditions
- 哪些结论只在特定 benchmark / metric / hardware / setting 下成立。

## Not Directly Comparable
- 哪些论文或结果不能直接放在同一比较轴上。

## Evidence Links
- 链接到支撑本页判断的 `evidence/*` 与必要的 `papers/*`。

## Open Questions
- 写出仍未解决但不构成当前结构性阻塞的问题。

## Gate Check
- `required_sections_complete: yes/no`
- `evidence_links_present: yes/no`
- `unscoped_comparative_claims: 0/nonzero`
- `boundary_conditions_present: yes/no`
