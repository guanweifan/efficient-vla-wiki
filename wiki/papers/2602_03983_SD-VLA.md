## Claim
- 这篇论文要解决的是：VLA 处理多帧历史时会遇到两类问题，一是多帧拼接导致 context 迅速膨胀，二是每一步都重算全部视觉 token 的 KV cache 会造成很高的推理开销；因此现有 VLA 很难既保留长时记忆，又保持推理效率。
- 作者提出 `SD-VLA`，核心是 `static-dynamic disentanglement`：把视觉 token 拆成具有不同时间持久性的 static tokens 与 dynamic tokens。static tokens 跨多个 timestep 只保留一份，dynamic tokens 才随时间拼接；同时通过 trainable `recache gate` 决定 static tokens 何时复用、何时刷新，从而把长时记忆建模与 KV-cache 复用统一起来。
- 论文主张：VLA 轨迹中的大量视觉信息在时间上是静态或缓慢变化的，若显式建模这种 temporal persistence，就能同时解决 long-horizon temporal context 与 efficient inference 两个瓶颈。
- headline numeric claims 包括：
  - 在作者提出的 memory-dependent benchmark 上，success rate 绝对提升约 `39.8%`。
  - 在 SimplerEnv 上，相比 base VLA 约有 `4.9%` success-rate gain，并达到约 `2.26×` inference speedup。
  - 在 LIBERO 上，相比 base model 约有 `0.7%` success-rate gain，并达到约 `1.70×` speedup。

## 待核点
- `39.8% absolute improvement` 来自作者新建的 memory-dependent benchmark，而不是 SimplerEnv / LIBERO；后续引用时要明确 benchmark 范围。
- `2.26×` 与 `1.70×` 分别对应 SimplerEnv 与 LIBERO，不应混成一个统一 speedup 数值。
- 当前主文里最容易直接抓到的是 SimplerEnv 与 memory benchmark 的表格结果；LIBERO 的更细指标若后续需要精确数值，可能还要继续回表格或附录核定。
- SD-VLA 的收益建立在 static token 可复用这一任务结构假设上；若后续推广到视觉变化极快、场景静态成分很弱的任务，需要保留适用范围 caveat。
