## Claim

页面定位：这是一篇 **fast-slow unified reasoning VLA** 论文；核心贡献是用 Mixture-of-Transformers 把 slow reasoning、fast reasoning 和 action diffusion 放进同一架构里，从而在 language steerability 与 reasoning latency 之间取得折中，而不是一般性的 manipulation benchmark 论文。

MoTVLA 的核心主张不是单纯“把 reasoning 接进 policy”，而是要在一个统一架构中同时保住 language steerability 与 inference efficiency。论文认为，现有路线要么不显式生成 reasoning，因此语言引导不足；要么把 reasoning 作为 next-token 生成结果再喂给 diffusion policy，导致推理延迟过高。为了解这个矛盾，MoTVLA 提出一种 Mixture-of-Transformers (MoT) 式的 VLA：用 generalist 负责 slow reasoning，用与其共享全局注意力的 domain expert 负责 fast reasoning，再把 fast reasoning 生成的 motion decomposition 与视觉 / 机器人状态一起送入 action expert（DiT）生成动作。

更稳的 headline 应拆开理解：`fast reasoning` 的质量与 token accuracy 主要来自 `Table 1`；manipulation performance 来自 `Table 2`；language steerability 在 `table bussing` 上主要还是 qualitative evidence；reasoning latency 则来自附录 `Table 4`，且主实验使用的是 `MoTVLA-14B`，`1B/0.5B` 主要用于 latency illustration。更准确的写法是：**MoTVLA 在统一架构里同时改善 reasoning latency、language-conditioned behavior 和 manipulation performance，但这些优势来自不同实验层面，不应压成单一 bundled superiority claim。**

显式 caveat：MoTVLA 的 headline 优势同样是组合性的。reasoning latency、NLP/VQA 表现、simulation manipulation、real-world manipulation、以及 table bussing 里的 language steerability，并不来自同一张表；此外论文明确说明主要定量结果使用的是 `MoTVLA-14B`，而 `1B/0.5B` 缩放结果主要用于 latency illustration，不应直接和主结果混写。

## Methodology Index

- **Mixture-of-Transformers reasoning backbone**：generalist 与 domain expert 共享 global attention，但在输出功能上分化为 slow reasoning 与 fast reasoning。
- **Decomposition-Composition-Decomposition**：输入按 modality 分解，经过统一 global self-attention 组合，再在输出端按功能重新分解成 general reasoning 与 domain reasoning。
- **Fast–slow unified reasoning**：slow reasoning 走标准 next-token prediction；fast reasoning 走 token-wise prediction，用更低延迟生成 domain-specific motion decomposition。
- **Action expert conditioned on motion decomposition**：将 fast reasoning 的 motion decomposition、视觉状态和 robot state 一起送入 diffusion transformer，形成 language-steered policy。
- **Dual-mode inference**：既支持 dialogue mode（只回答，不动作），也支持 action mode（执行任务），用来保证语言响应与动作执行边界清晰。
- **Two-stage curriculum / preserved general intelligence**：强调在学到 domain expertise 和 policy 的同时，不破坏 pretrained VLM 的 general intelligence。
