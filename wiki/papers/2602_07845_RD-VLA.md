## Claim

RD-VLA 的核心命题是：VLA 的 test-time compute scaling 不一定要通过显式 reasoning token 自回归展开来实现，更合适的做法是把“思考”移到动作头内部的 latent recurrent refinement 里，让模型在 constant-memory 条件下按任务难度动态增加推理深度。论文认为，现有 reasoning-centric VLA 往往把推理外显成 token generation，这会在连续动作场景里带来明显的 decoding 开销和 memory cost；而 RD-VLA 通过 weight-tied recurrent action head、latent scratchpad refinement 和 uncertainty-based adaptive stopping，把推理深度变成一个可在推理时弹性伸缩的内部变量。作者 headline claim 是：相较先前 reasoning-based VLAs，RD-VLA 可实现最高 `80x` inference speedup；在 LIBERO 上固定迭代版本达到 `93.0%` 成功率，adaptive 版本为 `92.5%`；在 CALVIN ABC→D 上达到 `3.39` average chain length 和 `45.3%` task-5 success。

显式 caveat：`80x`、`93.0%`、`92.5%`、`3.39`、`45.3%` 来自不同比较对象、不同 benchmark 和 fixed/adaptive 两种不同推理配置，不应被读成一个统一实验设置下的单一优势数字。论文真正更稳的主张是：**latent recurrent reasoning + adaptive stopping** 让 VLA 可以在不走显式 token reasoning 的情况下获得更好的 test-time compute elasticity。

## Methodology Index

- **Latent iterative reasoning**：把推理放进动作头内部的 latent recurrent refinement，而不是输出显式 reasoning token。
- **Prelude / Recurrent Core / Coda**：动作头分成 grounding、weight-tied recurrent refinement、action decoding 三段。
- **Weight-tied recurrent block**：同一 recurrent core 反复迭代，支持 arbitrary inference depth，同时保持较低参数与 memory 成本。
- **Latent scratchpad refinement**：从噪声 latent state 出发逐步收敛到动作相关表示，而不是逐 token 展开 reasoning trace。
- **TBPTT training**：用 truncated backpropagation through time 训练 recurrent action head。
- **Adaptive stopping via latent convergence**：根据相邻迭代间的 KL / convergence 信号提前退出，实现 task-dependent compute allocation。
