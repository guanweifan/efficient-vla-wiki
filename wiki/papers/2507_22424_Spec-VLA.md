## Claim
- 这篇论文要解决的是：Vision-Language-Action (`VLA`) 模型的自回归 action-token 生成较慢，而直接套用通用 speculative decoding 往往收益有限，因为 VLA 的 action prediction 更难、greedy acceptance 过于严格。
- 作者提出 `Spec-VLA`，把 speculative decoding 引入 VLA 推理，并在标准 draft-and-verify 框架上加入面向 action token 的 `relaxed acceptance` 机制：只要候选 token 与验证模型输出在相对距离上足够接近，就允许接受，而不再要求严格相等。
- 论文主张：VLA 场景下真正限制 speculative decoding 收益的，不只是 draft model 质量，还包括 action token 离散化后“近似正确但被严格拒绝”的情况；因此 acceptance 规则本身需要针对动作空间改造。
- headline numeric claims 包括：
  - relaxed acceptance 将 acceptance length 从约 `2.10` 提升到 `2.94`。
  - 相比 OpenVLA 的自回归解码，速度提升约 `1.22×` 到 `1.42×`。
  - 相比非 relaxed 的 speculative decoding，acceptance length 进一步提升约 `25%` 到 `44%`，同时 success rate 基本保持。

## 待核点
- `1.22×` 到 `1.42×` 的速度提升是 LIBERO 各 task suite 上的范围，不是单一无条件数值；后续引用时需要保留 benchmark-dependent caveat。
- “without compromising success rate” 更准确地说是：在主实验设置下 success rate 大体保持，但具体 suite 上仍有小幅波动；后续不能写成绝对无损。
- 论文的验证主要基于 OpenVLA-style 离散 action token 预测；若后续要把结论推广到更一般的连续控制或不同 tokenization 方案，需要保留适用范围。
- 训练与推理细节部分依赖 OpenVLA verification model + Eagle-style tree decoding；后续若写成通用 VLA speculative decoding recipe，需明确哪些设计是框架性的，哪些是当前实现选择。
