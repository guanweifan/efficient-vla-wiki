## Claim
- 这篇论文要解决的问题是：现有 VLA visual token pruning 方法大多延续了 **Perception-First** 偏置，主要根据 semantic saliency 或简单 temporal cue 做裁剪，因此容易在任务早期把“视觉上不显眼但对物理操作关键”的结构性区域提前裁掉。作者据此提出 **VLA-IAP**，把 pruning 从被动的 perception filtering 改成 **Interaction-First** 的显式交互对齐机制。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 1-3 页 Abstract、Introduction、Fig. 1。
- 方法上的主张是一个 **training-free** 的 token pruning 框架，由两部分构成：一是 **Geometric Prior Mechanism**，通过 edge enhancement 保留结构锚点；二是 **Interaction-Aligned Dynamic Strategy**，根据 semantic-motion IoU 在 conservative pruning 和 aggressive pruning 之间动态切换。其关键 framing 不是“更好的注意力裁剪”，而是借助几何先验和交互状态判断，让 pruning 强度与 manipulation phase 对齐。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 2-5 页 Fig. 1、Fig. 2、Sec. 3。
- 论文保留了多组 headline numeric claims：在 **LIBERO** 上实现 **97.8% success rate with 1.25× speedup**；在更激进设置下可达到 **1.54× speedup** 且维持与 unpruned backbone 可比的性能；real-world 部署中则报告 **up to 1.48× inference acceleration** 并同时提升 average manipulation success rate。这里的 caveat 是：这些数字对应不同 token retention ratio、不同 benchmark 与不同 backbone，不能在 `L1` 中被压成一个统一无条件的 superiority claim。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 1 页 Abstract；第 9-11 页 Table 2、Table 3、runtime / real-world analysis。
- 从当前证据看，VLA-IAP 更适合被视为一篇 **interaction-aligned, training-free visual token pruning** 论文：其价值不在于重新训练 backbone，而在于通过几何先验、IoU 门控和 phase-aware pruning 逻辑，使压缩过程更贴近 embodied manipulation 的物理约束。来源：[[raw/2603_22991_VLA-IAP.pdf]]，第 3-5 页 Sec. 3；第 15-16 页 Conclusion / appendix qualitative analysis。
