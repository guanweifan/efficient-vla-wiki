## Claim
- 页面定位：这是一篇 **unified multimodal parallel generation for VLA** 论文；它的核心贡献是用共享离散 token space 同时监督 text/image/action，并对三种模态采用不同的并行解码策略。
- 这篇论文要解决的是：统一型 VLA 如果继续沿用 autoregressive text generation，再分别为 image / action 设计 diffusion 或 re-mask 生成，会导致 attention 机制与训练流程复杂；而 fully autoregressive 的统一生成又会让 action inference 过慢。
- 作者提出 `MM-ACT`，把 text、image、action 统一到共享离散 token space 中，并采用 `parallel decoding` 做三模态生成：text 和 image 采用 re-mask parallel decoding，action 采用 one-step parallel decoding，从而在统一建模的同时兼顾动作推理效率。
- 论文主张：在共享 multimodal context 下联合监督 task planning、future image prediction 和 action chunk，可以通过 `Context-Shared Multimodal Learning` 反哺 action generation；跨模态联合训练比 action-only 学习更强，尤其对长时序规划和双臂操作有增益。
- headline 数字需要拆开理解：
  - `96.3%` 对应 `LIBERO`；
  - `72.0%` 对应 `Franka` real-robot 三任务；
  - `52.38%` 对应 `RoboTwin2.0` 八个双臂任务；
  - `+9.25%` 则是相对 `RoboTwin` action-only baseline 的 multimodal co-training gain，不是对所有 benchmark 的统一增益。
- 更稳的主张是：`MM-ACT` 通过共享离散 token space 和 modality-specific parallel decoding，把 planning、future image prediction 与 action generation 联合进同一训练框架；其价值既在 unified generation，也在 context-shared multimodal learning 对 action generation 的反哺。

## 待核点
- `96.3%` / `72.0%` / `52.38%` 分别来自 LIBERO、Franka real-robot、RoboTwin2.0，不应混成一个统一 benchmark headline。
- `+9.25%` 是相对于 RoboTwin action-only baseline 的 multimodal co-training gain，不是对所有 benchmark 的统一增益。
- 论文同时使用两种 parallel decoding 策略：text/image 用 re-mask，action 用 one-step；后续若写成“统一 parallel decoding”，应保留 modality-specific decoding 差异。
- `40 Hz (5 Hz per action chunk)` 来自 action chunk size=8 的 one-step parallel decoding 设计选择，属于特定推理配置，不应无条件外推到所有部署环境。
- 若后续把它并入一般 `unified VLA` 路线，需要保留“统一的是 token space 与训练框架，而不是所有模态都用同一 decoding 策略”这一边界。
