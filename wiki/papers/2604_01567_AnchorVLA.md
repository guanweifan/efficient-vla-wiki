## Claim
- 页面定位：这是一篇 **anchored diffusion for mobile manipulation** 论文；它的主贡献是把 diffusion sampling 限制在 anchor trajectory 附近，并用轻量 residual correction 做 test-time self-correction，而不是继续从纯噪声做完整 denoising。
- 这篇论文要解决的是：mobile manipulation 的动作分布天然多模态，deterministic L1 regression 容易平均化失效；但 full diffusion denoising 在控制时又太慢，而且 chunked execution 会积累 drift。来源：[[raw/2604_01567_AnchorVLA.pdf]]，Abstract；Introduction。
- `AnchorVLA` 的核心机制是 anchored truncated diffusion：先从 demonstration 中提取 representative anchor trajectories，再围绕这些 anchor 做局部 truncated denoising；执行时再加一个 residual correction module 做高频 test-time self-correction。来源：[[raw/2604_01567_AnchorVLA.pdf]]，Introduction；方法部分。
- headline 数字需要拆开理解：论文报告 `64.0%` ManiSkill-HAB average success、长 horizon 下 `61.5%` success、以及在某个 anchored truncation setting 下 `60.6 Hz -> 89.8 Hz`、约 `1.48×` speedup；这些数字对应不同实验层级与设置，不能直接混成统一结论。来源：[[raw/2604_01567_AnchorVLA.pdf]]，实验部分。
- 更稳的主张是：`AnchorVLA` 通过“anchor-guided initialization + truncated diffusion + test-time correction”在 mobile manipulation 中同时保留 multimodal action generation 与低延迟执行。
