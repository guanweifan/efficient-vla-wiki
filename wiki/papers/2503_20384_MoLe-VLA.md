## Claim
- 页面定位：这是一篇 **dynamic LLM layer activation / sparse routing for VLA** 论文；核心贡献是把 layer 当作 expert 做 Mixture-of-Layers 路由，而不是 early exit 或 token pruning。
- 这篇论文提出 **MoLe-VLA**（简称 **MoLe**），目标是通过**动态 layer activation** 提升 VLA 在机器人场景中的效率，而不是简单做 early exit 或 token pruning。作者的问题定义是：很多 LLM layers 在 robotics tasks 中存在冗余，但直接砍掉后层会损失对下游任务更关键的语义信息。来源：[[raw/2503_20384_MoLe-VLA.pdf]]，第 1-2 页 Abstract、Introduction。
- 核心主张是：把每个 LLM layer 视作一个 expert，构建 **Mixture-of-Layers** 机制，在输入端通过 router 动态选择当前任务最相关的 layers，以此在保持甚至提升 robotic performance 的同时显著降低计算成本。来源：[[raw/2503_20384_MoLe-VLA.pdf]]，第 1-3 页 Abstract、Introduction；第 4 页 Fig. 2。
- 为了让 layer-skipping 真正适配 embodied setting，作者引入两个关键组件：
  - **STAR**（Spatial-Temporal Aware Router）：同时利用视觉空间信息和语言时间信息做 layer selection
  - **CogKD**（cognition self-knowledge distillation）：用 full-layer teacher 给 sparse student 补回被跳层削弱的 cognition ability  
  来源：[[raw/2503_20384_MoLe-VLA.pdf]]，第 1-2 页 Abstract / contributions；第 2-5 页 method。
- 作者给出的 headline numeric claim 需要拆开理解：在 **RLBench + real-world** 实验中，MoLe-VLA 相比 baseline 可实现**最多 ×5.6 的 LLM computational cost reduction**，同时在十个任务上的**平均成功率提升最多 8%**。更稳的写法是：这些 headline 来自不同 baseline、任务集合与 evaluation layer，不应压成一组统一 operating point。来源：[[raw/2503_20384_MoLe-VLA.pdf]]，第 1 页 Abstract；第 2 页 contributions；第 6-8 页 tables / evaluation。
