## Claim
- 这篇论文要解决的是：现有 VLA 往往直接把视频扩散模型当作通用视觉条件器使用，却没有充分利用其中显式的空间边界与跨帧运动变化信息，导致视觉表征虽强，但对机器人动作生成的转化效率和实时性不足。
- 核心主张是：如果把视频扩散模型中的 foreground boundary 与 inter-frame motion variation 显式提炼出来，再异步注入一个更快的 DiT 动作专家，就能把 VDM 的感知优势转化成更强的操作策略，同时避免直接使用高维原始 diffusion token 带来的低效条件化。
- 作者提出 `Video2Act`，将视频扩散模型作为低频更新的感知 `System 2`，将 DiT action decoder 作为高频动作 `System 1`，并通过 `Spatial Filtering Operators` 与 `FFT` 提取 spatio-motional signals。论文 headline claim 是：相对先前 SOTA VLA 方法，在 simulation average success rate 上提升 `8.9%`，在 real-world average success rate 上提升 `21.7%`；详细实验段落中给出的 real-world 平均成功率为 `73.3%`。来源：[[raw/2512_03044_Video2Act.pdf]]，第 1 页摘要；第 6-7 页 real-world results。

- 方法总览：[[raw/2512_03044_Video2Act.pdf]] 第 3-5 页 Fig. 3 与 Section 3。这里定义 `Spatial Filtering Operators`、`FFT`、异步 dual-system 更新方式，以及为什么只把结构化 spatio-motional signals 注入 DiT。
- 模拟主结果与 token 长度选择：[[raw/2512_03044_Video2Act.pdf]] 第 5-6 页 simulation results 与相关表格。这里可回查 simulation success rate、与 prior VLA 的对比，以及 `128 / 256 / 512 / 1024` token length 的 tradeoff。

## 待核点
- `8.9%` 与 `21.7%` 是相对 prior SOTA 的平均提升幅度，不是论文自己的绝对 success rate；后续若写 headline，需要避免和绝对 `73.3%` 混口径。
- 论文同时出现了 `ALOHA dual-arm robot` 与 `Agilex Robot / Cobot Magic` 的表述，平台命名需要后续统一核对，避免把不同 real-world setup 混成同一实验描述。
- 论文强调 `real-time action generation frequency`，但主文里更明确给出了 `587.9 ms` cold-start latency，而非一个统一稳定的控制频率 headline；后续若写“实时”需要保留这个 caveat。
- `Video2Act` 的主命题既可以写成“VDM-driven dual-system policy”，也可以写成“spatio-motional representation extraction for action generation”；后续需要决定页面主轴。
