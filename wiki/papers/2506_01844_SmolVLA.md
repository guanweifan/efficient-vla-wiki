## Claim
- 这篇论文提出 **SmolVLA**，核心定位是一个 **small, efficient, community-driven** 的 VLA：目标不是追求最大模型规模，而是把训练、部署与开源可复现性压到更普适的硬件与社区数据条件下。
- 作者的主命题包含三部分：  
  1. **轻量架构**：小型预训练 VLM + action expert，配合 skip VLM layers、较少视觉 token、交替 cross/self-attention 等设计；  
  2. **社区数据预训练**：不依赖昂贵专有机器人数据，而是用公开 community-collected datasets 做预训练；  
  3. **异步推理栈**：把 observation processing / action prediction 与 action execution 解耦，以提升实际控制响应性。  
- 论文在 abstract 中给出的 headline 主张是：SmolVLA 可以在 **single GPU** 上训练、在消费级 GPU 甚至 **CPU** 上部署；借助异步推理与 chunked action generation 获得更高 control rate；尽管模型更小，性能仍与 **约 10× larger** 的 VLA 可比。  
  来源：[[raw/2506_01844_SmolVLA.pdf]]，第 1 页，Abstract。
- 论文在贡献部分进一步明确了训练数据规模：SmolVLA 端到端预训练使用 **22.9K episodes / 10.6M frames**，来自 **481** 个公开 community datasets；作者把这一规模描述为相较 prior art 至少低一个数量级。  
  来源：[[raw/2506_01844_SmolVLA.pdf]]，第 2 页，贡献列表；第 5 页，Table 1。

## Methodology Index
- compact pretrained VLM + action expert
- skip top VLM layers
- minimal visual tokens
- small pretrained VLM backbone
- alternating cross-attention and self-attention in action expert
- flow matching for chunked low-level actions
- community-driven pretraining
- asynchronous inference stack
- decoupled action execution vs perception/prediction
- affordable / low-cost robot emphasis
