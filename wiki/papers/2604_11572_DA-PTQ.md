## Claim
- 页面定位：这是一篇 **drift-aware post-training quantization / low-bit VLA deployment** 论文；它的核心贡献不是新的 action generator，而是把 VLA 低比特量化失败解释为 sequential control 中的 trajectory-level drift，并据此设计校准与混合精度分配。
- DA-PTQ 的核心判断是：直接把已有 PTQ 套到 VLA 上会在 vision-language-to-action interface 引入量化扰动，这些扰动会在连续控制中逐步放大，形成 kinematic drift。来源：[[raw/2604_11572_DA-PTQ.pdf]]，Abstract、Introduction、Figure 1。
- 方法由两个主要组件构成：**Cross-Space Representation Compensation** 用于缓解 multimodal representation 与 action space 之间的结构性失真，**Motion-Driven / Drift-Aware Mixed-Precision Allocation** 则按 trajectory-level motion error 分配 bit-width。来源：[[raw/2604_11572_DA-PTQ.pdf]]，Abstract、Figure 2、method section。
- headline 结果需要拆开理解：
  - `42.5% memory reduction` 与 `54.8% inference speedup` 是 SimplerEnv efficiency-performance trade-off 表中的系统性效率口径；
  - WidowX 与 Google Robot 的 success rate 是控制质量口径；
  - “reduces kinematic drift” 是方法 failure-mode 解释，不等同于任意低比特设置下都保持全精度能力。
- 更稳的主张是：DA-PTQ 把 VLA PTQ 从静态量化误差问题推进到 **trajectory-level quantization drift** 问题，适合作为 low-bit / deployment-oriented VLA efficiency 的新增证据。
